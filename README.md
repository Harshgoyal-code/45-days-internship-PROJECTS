# 🚀 45 Days Internship Projects

This repository contains the machine learning and Flask projects completed during my 45-day internship. Each project demonstrates the implementation of machine learning models integrated with a Flask web application.

## 📂 Projects

### 1. Female Diabetes Predictor
- Predicts diabetes risk based on medical parameters.
- **Technologies:** Python, Flask, Scikit-learn, HTML, CSS

### 2. Fraud Detection System
- Detects whether a financial transaction is fraudulent or legitimate using a trained machine learning model.
- **Technologies:** Python, Flask, Scikit-learn, HTML, CSS

### 3. Farmer Crop and Fertilizer Recommendation
- Recommends suitable crops based on soil and environmental conditions.
- Suggests the appropriate fertilizer based on crop and soil information.
- **Technologies:** Python, Flask, Scikit-learn, HTML, CSS

## 🛠️ Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- NumPy
- HTML5
- CSS3
- Jupyter Notebook
